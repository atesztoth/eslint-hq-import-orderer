# eslint-plugin-eslint-hq-import-orderer

This plugin will order your imports to look so badass you&#39;re gonna tatto it on yourself!

## Installation

You'll first need to install [ESLint](http://eslint.org):

```
$ npm i eslint --save-dev
```

Next, install `eslint-plugin-eslint-hq-import-orderer`:

```
$ npm install eslint-plugin-eslint-hq-import-orderer --save-dev
```


## Usage

Add `eslint-hq-import-orderer` to the plugins section of your `.eslintrc` configuration file. You can omit the `eslint-plugin-` prefix:

```json
{
    "plugins": [
        "eslint-hq-import-orderer"
    ]
}
```
